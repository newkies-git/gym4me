# Gym4me Agent Skills

프로젝트에 맞춘 Cursor Agent 스킬 정의입니다.

## 스킬 목록

| 스킬 | 용도 |
|------|------|
| **frontend_agent** | Vue 3, TypeScript, Pinia, Firebase 클라이언트, 컴포넌트·뷰 개발 |
| **firebase_agent** | Firestore, Auth, Cloud Storage, Cloud Functions (Node.js) |
| **composable_agent** | Composable 추출·생성, 로직 캡슐화 |
| **test_agent** | Vitest, Vue Test Utils, 단위 테스트 |
| **page-flow-analyzer** | 단일 Vue 페이지의 모달/팝업 open-close 플로우 분석 |

## 기술 스택 (참고)

- **Frontend**: Vue 3, Vite 7, TypeScript, Pinia, Firebase SDK
- **Backend**: Firebase (Firestore, Auth, Storage, Cloud Functions)
- **Test**: Vitest, @vue/test-utils

이 프로젝트는 **Spring Boot, Java, GraphQL**을 사용하지 않습니다.
