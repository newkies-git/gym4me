# Berry Vue Admin Template 

## 🚀 Getting Started
Follow these steps to set up the project:

1. Navigate to your root folder
```
c:> cd berry-vuertify-vuejs
```

2. Install packages

```
npm install
```

3. Run project

```
npm run dev
```
