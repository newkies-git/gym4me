**작성 목적**: Gym4me 서비스의 비전, 목표, 핵심 기능 및 로드맵 정의

## 1. 서비스 개요

### 1.1 서비스 비전

**Gym4me**는 헬스장 관리, 트레이너-회원 매칭, PT(개인 트레이닝) 일정 및 운동 기록 관리를 통합하는 **헬스장 운영·관리 플랫폼**입니다.

- **타깃 사용자**: 헬스장 운영자, 매니저, 트레이너, 회원(트레이니)
- **핵심 가치**: 일정·수강권·운동 기록의 디지털화로 운영 효율과 회원 경험 향상

### 1.2 서비스 목표

| 목표 | 설명 |
|------|------|
| **운영 효율화** | 지점(Gym)·트레이너·회원·수강권을 한 곳에서 관리 |
| **PT 일정 관리** | 개인 PT, 그룹 클래스, 강좌 일정 통합 및 승인/완료 워크플로우 |
| **운동 기록** | 세션별 운동 로그, 바디 프로필, 과거 기록 검색 |
| **확장성** | PWA·모바일 앱(Capacitor) 지원, 다국어(i18n) 기반 |

## 3. 핵심 기능 영역

### 3.1 인증 및 사용자 관리

- 이메일 기반 Firebase Authentication
- 회원가입: 비밀번호 확인, 이용약관·개인정보 동의
- 최초 로그인(트레이니): 이름·전화·Gym 필수 입력 후 서비스 이용
- 역할별 홈 화면 및 메뉴 노출

### 3.2 홈 (역할별 대시보드)

- **SITE_ADMIN**: Gym 관리·매니저 관리 링크, StatCard
- **MANAGER**: 트레이너 관리·지점 트레이니 링크, StatCard
- **TRAINER**: 담당 트레이니·그룹 클래스 관리
- **MEMBER**: 잔여 세션·만료일·다가오는 세션, 과거 운동 검색, 캘린더·바디 프로필 링크
- **OBSERVER**: 안내 배너, 테스트 수강권 구매(시뮬레이션)

### 3.3 일정 관리 (Calendar)

- 개인 PT, 클래스, 강좌 일정 통합 표시
- 2주 단위 그리드, 주간 이동·오늘 버튼
- 상태: PENDING → APPROVED / REJECTED / COMPLETED / CANCELLED
- 세션 완료: 디지털 서명, 수강권 차감, 운동 기록 연동

### 3.4 강좌 (Courses)

- 강좌 CRUD (트레이너 이상)
- Gym·참석자 선택, 트레이니 신청·트레이너 승인
- 강좌 일정이 캘린더에 자동 반영

### 3.5 트레이니·클래스 관리 (Trainer)

- 담당 트레이니 목록, 이메일 검색 등록
- 수강권·만료일 수정
- 그룹 클래스 생성·초대·트레이니 관리
- 클래스별 캘린더 뷰

### 3.6 운동 기록 및 신체 데이터

- 세션별 운동 종목·세트·횟수·무게 기록
- 바디 프로필(체중·체지방 등) Chart.js 시각화
- 과거 운동 검색 후 "오늘 반복 추가"

### 3.7 Gym·트레이너·직원 관리

- **Gym**: 지점 등록·수정·삭제, PRIMARY/VICE 매니저 지정
- **트레이너**: 역할 부여, 소속 Gym 지정
- **지점 트레이니**: 해당 Gym 회원 목록·수강권 조정
- **매니저/직원**(SITE_ADMIN): AdminLayout 하위 managers, staff 관리

### 3.8 기타 기능

- **기구 사용법(Tool Usage)**: 영상·이미지 CRUD, 권한별 공개/비공개
- **트레이너 프로필**: 소개·전문분야·경력 편집
- **테마**: 8종 프리셋, localStorage 저장
- **다국어**: 한국어/영어 (vue-i18n)
