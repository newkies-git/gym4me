# Gym4me 화면 정의서 (Wireframe / UI Flow)

**문서 버전**: 1.0  
**최종 수정일**: 2025-03-19  
**작성 목적**: 화면 구성, 라우트, UI 흐름 및 주요 컴포넌트 정의

---

## 1. 화면 구조 개요

### 1.1 전체 라우트 맵

```
/ (랜딩)
├── /auth (로그인·회원가입)
├── /terms (이용약관)
├── /privacy (개인정보처리방침)
│
├── /complete-profile (최초 프로필 완료) [인증 필수]
│
├── /home (역할별 홈) [인증 필수]
│   ├── SITE_ADMIN → SiteAdminHome
│   ├── MANAGER → ManagerHome
│   ├── TRAINER → TrainerHome
│   ├── MEMBER → TraineeHome
│   └── OBSERVER → ObserverHome
│
├── /dashboard → /home 리다이렉트
│
├── /calendar (일정) [인증 필수]
├── /profile (바디 프로필) [인증 필수]
├── /user-info (계정 설정) [인증 필수]
├── /settings → /user-info 리다이렉트
│
├── /courses (강좌) [인증 필수]
├── /tool-usage (기구 사용법) [인증 필수]
│
├── /trainer-profile (트레이너 프로필) [인증+트레이너]
├── /manage-trainers (트레이너 관리) [인증+매니저]
├── /manage-gym (Gym 관리) [인증+매니저]
├── /gym/trainees (지점 트레이니) [인증+매니저]
├── /gym/members → /gym/trainees 리다이렉트
│
└── /admin (관리자) [인증+SITE_ADMIN]
    ├── /admin/managers (매니저 관리)
    └── /admin/staff (직원 관리)
```

## 2. 화면별 상세 정의

### 2.1 공개 화면 (인증 불필요)

#### 2.1.1 랜딩 (`/`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | HomeView.vue |
| **설명** | 서비스 소개, 로그인 시 `/home` 리다이렉트 |
| **주요 요소** | 로고, CTA 버튼(로그인/회원가입) |

#### 2.1.2 인증 (`/auth`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | AuthView.vue |
| **설명** | 로그인·회원가입 탭 |
| **주요 요소** | 이메일 입력, 비밀번호 입력, ID 기억 체크, 회원가입 시 이용약관·개인정보 동의 |

#### 2.1.3 이용약관 (`/terms`), 개인정보처리방침 (`/privacy`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | TermsView.vue, PrivacyView.vue |
| **설명** | 정적 페이지 |

### 2.2 인증 후 공통 화면

#### 2.2.1 프로필 완료 (`/complete-profile`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | CompleteProfileView.vue |
| **조건** | needsMemberProfile=true (트레이니 최초 로그인) |
| **주요 요소** | 이름, 전화번호, Gym 선택 필드 |
| **완료 후** | `/home` 이동 |

#### 2.2.2 홈 (`/home`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | DashboardView.vue |
| **역할별** | SiteAdminHome, ManagerHome, TrainerHome, TraineeHome, ObserverHome |
| **공통** | GNB, 로그인 시 소속 Gym명 표시 |

**역할별 홈 구성요소:**

| 역할 | StatCard | 주요 링크/컴포넌트 |
|------|----------|-------------------|
| SITE_ADMIN | 2개 | Gym 관리, 매니저 관리 |
| MANAGER | 2개 | 트레이너 관리, 지점 트레이니 |
| TRAINER | - | TraineeManager, ClassManager |
| MEMBER | 3개(잔여·만료·다가오는 세션) | PastExerciseSearch, 캘린더·바디 프로필 |
| OBSERVER | - | 안내 배너, 테스트 수강권 구매 |

#### 2.2.3 일정 (`/calendar`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | CalendarView.vue |
| **Composable** | useCalendarPeriod |
| **주요 요소** | 2주 그리드, 이전/다음 주, 오늘 버튼, 일정 추가 버튼 |
| **모달** | AddScheduleModal, EventDetailsModal |
| **일정 타입** | Individual PT, Class, Course |

#### 2.2.4 바디 프로필 (`/profile`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | BodyProfileView.vue |
| **주요 요소** | ProfileChart, ProfileForm, ProfileHistory |
| **기능** | 체중·체지방 등 입력, Chart.js 시각화 |

#### 2.2.5 계정 설정 (`/user-info`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | UserInfoView.vue |
| **주요 요소** | 프로필 수정, 비밀번호 변경 |

#### 2.2.6 강좌 (`/courses`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | CourseListView.vue |
| **Composable** | useCourseList |
| **주요 요소** | 강좌 목록, 상세, 생성, 신청, 승인 |

#### 2.2.7 기구 사용법 (`/tool-usage`)

| 항목 | 내용 |
|------|------|
| **컴포넌트** | ToolUsageView.vue |
| **기능** | 영상·이미지 CRUD, 권한별 공개/비공개 |

## 3. UI 흐름

### 3.1 인증 플로우

```
[랜딩 /] → [로그인 /auth] → [인증 성공]
    → needsMemberProfile? 
        → Yes: /complete-profile → /home
        → No: /home
```

### 3.2 권한별 네비게이션

| 역할 | GNB 접근 가능 메뉴 |
|------|-------------------|
| SITE_ADMIN | 홈, 캘린더, 강좌, 프로필, 기구 사용법, 트레이너 프로필, Gym 관리, 관리자 |
| MANAGER | 홈, 캘린더, 강좌, 프로필, 기구 사용법, 트레이너 프로필, Gym 관리, 지점 트레이니 |
| TRAINER | 홈, 캘린더, 강좌, 프로필, 기구 사용법, 트레이너 프로필 |
| MEMBER | 홈, 캘린더, 강좌, 프로필, 기구 사용법 |
| OBSERVER | 홈 제한 |

### 3.3 일정 상세 플로우

```
[캘린더] → [일정 클릭] → [EventDetailsModal]
    → PENDING: [승인] / [거절(사유)]
    → APPROVED: [세션 완료] → [서명] → [수강권 차감] → [운동 기록]
```
